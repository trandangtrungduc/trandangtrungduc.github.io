This project is a Vibe Coding Web Application built with **TypeScript** on the frontend and **Python** on the backend, designed for ultra-fast AI image generation.

It leverages state-of-the-art models such as [Sana](https://github.com/NVlabs/Sana), capable of generating images in under **0.2 seconds per image**, and [Infinity](https://github.com/FoundationVision/Infinity) for high-quality and scalable generation.

The system supports advanced features including **ControlNet** for guided image generation, **Inpainting** for precise image editing, and **custom LoRA training** to adapt and personalize image styles. This enables users to quickly generate, refine, and customize visuals with high performance and flexibility.
