**Fast Game Asset Generation Application** is a _Vibe Coding Web Application_ built with **TypeScript** on the frontend and a **Python backend**, focused on ultra-fast AI-powered image generation for game development.

The system leverages cutting-edge models such as [Sana](https://github.com/NVlabs/Sana), capable of generating images in under **0.2 seconds per image**, and [Infinity](https://github.com/FoundationVision/Infinity) for scalable, high-quality generation. This combination enables near real-time asset creation, making it ideal for rapid iteration workflows.

Beyond speed, the platform provides advanced capabilities including **ControlNet** for guided generation, **Inpainting** for precise editing, and **custom LoRA training** to adapt and personalize styles. These features allow users to not only generate assets quickly, but also refine and control outputs with a high degree of flexibility.

Designed for developers and artists, the system supports a wide range of use cases—from **stickers and sprites** to **textures and visual assets**—streamlining the asset creation pipeline for games and interactive applications.
