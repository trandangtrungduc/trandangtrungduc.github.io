This project is part of a Vibe Coding Game series, developed using the **Pygame**. The core gameplay is inspired by the original [Shooter](https://www.codingwithruss.com/gamepage/Shooter/) game, with enhancements focused on integrating AI-driven interactions.

The in-game 2D AI character (design, generation and animation) is generated using the [PixelLab platform](https://www.pixellab.ai/), bringing a unique visual identity to the experience. Additionally, the project leverages advanced language and speech capabilities powered by **Qwen** from Alibaba, incorporating both LLM (language understanding) and TTS (text-to-speech) models to create a more immersive and dynamic gameplay experience.
