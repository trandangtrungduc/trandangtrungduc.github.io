**AI Character in Game** is part of a _Vibe Coding Game series_, exploring how AI-driven characters can enhance gameplay through interaction, personality, and dynamic behavior.

The 2D prototype is developed using **Pygame**, inspired by the original [Shooter](https://www.codingwithruss.com/gamepage/Shooter/) game, but extended with real-time AI interaction. The in-game character—designed, generated, and animated via [PixelLab](https://www.pixellab.ai/)—features a distinct pixel-art style and is powered by **Qwen** (LLM + TTS), enabling natural dialogue, voice output, and more immersive player engagement.

The project also extends into a 3D prototype, where characters are generated using the [Hunyuan 3D platform](https://3d.hunyuanglobal.com/) from Tencent. This version builds on the [Multiplayer Shooter Game](https://github.com/wass08/r3f-playroom-multiplayer-shooter-game), integrating AI-driven characters into a real-time multiplayer environment.

By combining AI-generated assets with language and voice capabilities, the project demonstrates how intelligent characters can move beyond scripted behavior to become interactive, expressive elements within both 2D and 3D game worlds.
