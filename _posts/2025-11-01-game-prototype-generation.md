This project is a _Vibe Coding Web Application_ built with **TypeScript** on the frontend and **Python** on the backend.

It integrates **Segment Anything** for rapidly extracting and recreating game assets, enabling efficient asset generation and reuse. Additionally, the system leverages **Google Gemini** to generate game logic and content directly from prompts, allowing users to quickly build complete mechanic game experiences using the created assets.

By combining automated asset processing with prompt-based game generation, this project streamlines the game development workflow and empowers rapid prototyping.
