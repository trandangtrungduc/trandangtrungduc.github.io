**AI Game Warehouse Application** is a _Vibe Coding Web Application_ built with **TypeScript** and powered by **Google Gemini**, designed as a hub for AI-generated interactive game experiences.

The platform acts as a **repository of playable AI-generated games**, where each entry is a self-contained application that runs directly in the browser. Beyond static gameplay, users can interact with AI in real time within each game, enabling dynamic narratives, adaptive mechanics, and more engaging experiences.

Rather than focusing on polished releases, the system is built for **rapid prototyping and experimentation**—making it easy to create, store, and showcase new ideas. It provides a lightweight environment for testing AI-driven gameplay concepts, iterating quickly, and sharing demos without complex setup.
