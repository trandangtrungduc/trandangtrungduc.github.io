**CronyClaw** is a _Vibe Coding Web & Desktop Application_ built with **TypeScript**, powered by a **Python backend**, and packaged with **Electron** for a seamless cross-platform desktop experience.

Inspired by and extending the original [OpenLLM Vtuber](https://github.com/Open-LLM-VTuber/Open-LLM-VTuber), the project brings to life an interactive **Live2D AI character** capable of real-time conversation, **voice cloning**, emotional expression, and contextual screen awareness. The system integrates multiple LLM and VLM technologies (such as Qwen, OpenAI, Gemini, and Ollama) alongside TTS/ASR pipelines to enable rich, multimodal interaction.

CronyClaw also features deep integration with [OpenClaw](https://github.com/openclaw/openclaw) via **OpenClaw Bridge**, allowing the AI to connect through direct **Auth Integration** or modular **Skill-based execution**. This enables the assistant to manage tasks (e.g., creating, starting, completing workflows) through a unified interface.

Users can interact using text or voice (**ASR**), while the AI dynamically composes and executes skills to perform complex actions, effectively turning the character into a capable, autonomous task operator.
