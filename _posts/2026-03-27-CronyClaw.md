This project is a *Vibe Coding Web & Desktop Application* built with **TypeScript**, featuring a **Python backend** and **Electron** for the desktop experience.

It extends the original [OpenLLM Vtuber](https://github.com/Open-LLM-VTuber/Open-LLM-VTuber) project to create an interactive **Live2D character** capable of real-time communication, voice cloning, emotional expression, and screen awareness.

The system integrates multiple LLM and VLM technologies, including **Qwen**, **OpenAI**, **Gemini**, **Ollama**, **IndexTTS**, and **Edge TTS**,... enabling rich multimodal interaction between the user and the AI character.

Additionally, it supports seamless integration with [OpenClaw](https://github.com/openclaw/openclaw), allowing users to manage tasks (e.g., starting, completing) through a unified interface. Users can interact via text messages or **ASR** using their voice, while the AI can combine different **skills** to execute tasks efficiently through OpenClaw.
